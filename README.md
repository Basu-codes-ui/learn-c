# Circle Area & Perimeter Program in C

This is a simple C program that calculates:

- Area of a Circle
- Perimeter (Circumference) of a Circle

## Formula Used

Area = π × r × r  
Perimeter = 2 × π × r  

## Concepts Used

- Macros (#define)
- User Input using scanf
- Floating point calculations
- Basic mathematical operations
